# Changelog

This changelog tracks changes to the public OpsHome Docker Probe examples and setup notes.

It is not the OpsHome NOC product release log. For product updates, see the Docs Release Log:

https://docs.opshome.run/release-notes.html

## 0.1.0

- Added initial Docker Compose example with placeholder token values.
- Added outbound-only security model notes.
- Added private HTTP, HTTPS, and TCP monitoring examples.
- Added troubleshooting notes for Docker Probe offline, invalid token, network unreachable, and Private Monitor failed cases.
